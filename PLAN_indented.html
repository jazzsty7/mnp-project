💬 카카오톡형 중매 협업앱

마담뚜 업무앱 PLAN

50~60대 여성 중매인이 기존 카카오톡 단체방에서 하던 업무를 그대로 유지하면서, 고객카드·소개하기·소개방·참여요청 관리만 추가한 모바일웹 기획안입니다.

1. 서비스 방향

새로운 CRM이 아니다

복잡한 관리자 시스템이 아니라, 카카오톡처럼 보이고 카카오톡처럼 쓰는 협업 메신저를 목표로 한다.

채팅방이 중심이다

고객은 채팅방 안에서 공유되고, 소개하기가 발생하면 요청함과 소개방으로 자연스럽게 분리된다.

버튼은 동사형으로

50~60대 사용자를 위해 공유하기, 소개하기, 수락하기, 거절하기처럼 행동이 바로 보이는 표현을 사용한다.

12. 개발 시 우선 적용 규칙

화면 이동만 구현

초기 HTML 프로토타입에서는 저장, 등록, 승인 로직을 만들지 않고 버튼 클릭 시 해당 화면으로 이동하는 것만 구현한다.

소스 가독성 유지

각 HTML은 들여쓰기와 개행을 적용해 초보자도 구조를 쉽게 확인할 수 있도록 작성한다.

50~60대 기준 UI

큰 글자, 큰 버튼, 4개 이하 하단 메뉴(채팅·내 고객·요청·더보기), 복잡한 용어 제거를 기본 원칙으로 한다.

카카오톡과 유사한 사용감

사용자가 “카톡이네, 그런데 고객카드와 소개방이 있네”라고 느끼게 하는 것이 목표다.

v9 화면 수정 규칙

  • 고객카드의 상세정보 / 소개하기 버튼은 고객 이름 시작 위치와 같은 라인에서 시작하고, 남은 가로 폭을 활용해 넓게 표시한다.
  • 단체방 하단에는 고객 공유용 둥근 + 버튼을 대화 영역에 배치한다.
  • 단체방과 1:1 대화방 입력창의 + 버튼은 사진 또는 파일 선택 용도로 사용한다.
  • 1:1 대화방 하단의 진행 버튼은 제거하고, 진행 상태는 상단 제목 우측 배지로 표시한다.
  • v14: 단체방 고객카드는 화면 우측을 넘지 않도록 최대폭을 제한하고, 1:1 대화방 말풍선은 중간에서 잘리지 않도록 줄바꿈/폭 규칙을 보정한다.
  • v17: 단체방 고객카드는 채팅 화면 안에서 가능한 최대 가로폭을 사용하되, 화면 우측으로 넘어가지 않도록 max-width: 100%와 box-sizing: border-box를 유지한다.
  • v18: 단체방 고객카드는 너무 꽉 차 보이지 않도록 가로폭을 약간 줄이고, 상대편이 올린 카드임을 알 수 있도록 좌측 정렬과 여백을 유지한다.

1:1 대화방 상태 최종 규칙

방장은 1:1 대화방에 고객을 공유한 회원입니다.

  • 진행중: 파란색 표시. 양쪽 채팅 목록에 모두 보입니다.
  • 종료중: 빨간색 표시. 방장이 아닌 회원이 나간 상태이며, 나간 회원 목록에서는 사라지고 방장 목록에는 남습니다.
  • 종료: 검정색 표시. 방장이 나간 상태이며, 양쪽 채팅 목록에서 사라지고 향후 PC 관리자 시스템에서 관리합니다.
  • v16: 단체방 고객카드는 내 고객 카드보다 약간 작게 보이되, 기존보다 가로폭을 넓혀 화면을 더 활용한다.
  • 단체방 고객카드는 화면 우측으로 넘어가지 않도록 최대폭 제한을 유지한다.

v12 적용 규칙

  • 채팅방과 내 고객 화면의 고객카드는 동일한 가로형 단일 카드로 사용한다.
  • 고객카드의 정보는 최소화한다: 이름, 상태/소개요청건수, 34세 · 서울 · 약사.
  • 고객 사진은 반명함판 비율을 유지하되, 가로형 카드 안에서 더 크게 보여준다.
  • 채팅방 하단 입력창은 카카오톡처럼 하나의 둥근 입력창 안에 좌측 + 버튼, 중앙 입력 영역, 우측 화살표 전송 버튼을 둔다.
  • 좌측 + 버튼은 사진 또는 파일 선택 용도다.

추가 수정 규칙

  • 채팅 화면은 상단에 단체방, 하단에 1:1 대화방을 구분해서 보여준다.
  • 소개방은 요청 메뉴가 아니라 채팅 메뉴의 1:1 대화방 영역에서 관리한다.
  • 요청 화면은 받은요청/보낸요청을 선택할 수 있게 하고, 보낸 요청 목록에는 버튼을 두지 않는다. 보낸 요청 상세화면 하단에서 취소한다.
  • 고객 추가와 상세정보 화면에는 반명함판(3.5 × 4.5cm) 비율의 증명사진 등록/변경 영역을 제공한다.
  • 요청 화면은 라디오 버튼이 아니라 상단 탭 UI로 받은요청/보낸요청을 전환한다.
  • 고객카드는 대화방 폭을 고려한 가로형 카드로 구성하고, 표시 정보는 “34세 · 서울 · 약사”처럼 최소화한다.
  • 고객 사진은 반명함판 비율을 유지하되, 대화방에서는 작은 세로형 썸네일로 배치한다.
  • 채팅방 목록에서 비회원이 비공개방을 누르면 참여요청 팝업을 먼저 보여준다.
  • 1:1 대화방 상단에는 방 제목 옆에 진행중/종료 상태를 표시한다.
  • 1:1 대화방 우측 끝에는 햄버거 메뉴를 둔다.
  • 햄버거 메뉴에는 채팅방 나가기를 표시한다.
  • 채팅방 나가기는 내 채팅 목록에서만 숨김 처리한다.
  • 방장이 나가면 방은 종료되며 양쪽 모두의 채팅 목록에서 보이지 않는다.
  • 상대방의 채팅 목록에서는 해당 1:1 대화방이 사라지지 않는다.

v13 채팅 메시지 정렬 규칙

  • 상대편 메시지는 좌측 정렬한다.
  • 내 메시지는 우측 정렬한다.
  • 고객카드는 단체방/내 고객에서 가로형 카드로 넓게 표시한다.

v20 반영 규칙

  • 내 고객에서 공유하기를 누르면 공유할 고객은 이미 정해져 있으므로 읽기 전용 텍스트로 표시하고, 올릴 채팅방만 선택한다.
  • 내 고객 화면은 여성/남성 각각 2~3명 샘플을 두고, 전체에서는 모든 고객카드를 노출한다.
  • 채팅 목록에서는 재혼 모임방만 참여 요청 팝업을 띄우고, 강남 전문직방은 바로 입장한다.
  • 1:1 대화방 목록에는 진행중/종료중/종료 샘플을 표시하고, 대화방 상세 상단 상태는 목록에서 클릭한 상태와 일치시킨다.

v21 수정 규칙

  • 내 고객 화면은 전체/여성/남성 탭으로 구분하고, 각 탭에서는 해당 고객 목록만 표시한다.
  • 1:1 대화방 목록에서 선택한 방의 제목과 상태는 대화방 상단 제목/상태와 반드시 일치한다.
